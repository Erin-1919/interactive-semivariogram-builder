Step 3. Fit a Theoretical Semivariogram Model

An empirical semivariogram is a scatter of observed γ values — for Kriging we need a smooth, mathematically defined curve.

Spherical
Best used when
  • Spatial similarity decreases steadily then stops beyond a certain distance
  • General-purpose environmental data
Exponential
Best used when
  • Nearby points may differ substantially
  • Patchy or irregular variables
  • Air pollution, contaminant spread
Gaussian
Best used when
  • Spatial processes change smoothly over space
  • Highly continuous surfaces
  • Temperature, groundwater surfaces

Step 6. Validation

Evaluate whether the kriging model performs well. A common approach is leave-one-out cross-validation:

Steps
  1. Remove one observed point
  2. Predict its value using the remaining points
  3. Compare predicted vs. observed value
  4. Repeat for all points
Metric Desired result
Mean Error Close to 0
Root Mean Square Error As small as possible
Mean Standardized Error Close to 0
Root Mean Sq. Standardized Error Close to 1
